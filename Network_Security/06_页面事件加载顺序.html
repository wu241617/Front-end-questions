页面加载过程

  • DNS域名解析
  • 建立TCP连接
  • 发起HTTP请求
  • 服务器处理请求
  • 返回响应结果
  • 关闭TCP连接
  • 浏览器解析HTML
    1. 形成DOM树
    2. 形成CSS样式树
    3. DOM树和CSS样式树组合形成Render Tree
  • 浏览器布局渲染
    1. Layout
    2. paint

DNS域名解析步骤

  • DNS域名解析都会有个DNS服务器将域名解析为ip地址，以便向正确的ip地址发送请求。
  • 判断当前页面DNS是否存在缓存
  • 查询本地DNS缓存
  • 请求local NDS Service
  • 根DNS服务器
  • 进行域名解析
  • 返回ip以及TTL值
  • DNS缓存
  • 返回解析结果给用户

建立TCP连接

  • 建立TCP连接，主要有数据封装和三次握手（建立连接）。
  • 发送HTTP请求的动作包括 GET / POST / OUT / DELETE / OPTIONS 等这几种
