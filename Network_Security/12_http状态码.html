HTTP1.1状态码20x,30x,40x,50x等个区间含义

  • 20x:成功处理请求
    1. 200成功
    2. 201已创建
    3. 202已接受
    4. 203非授权
    5. 204无内容
    6. 205重置内容
    7. 206部分内容
  • 30x:重定向请求
    1. 300多种选择
    2. 301永久移动
    3. 302临时移动
    4. 303查看其它位置
    5. 304未修改
    6. 305代理
    7. 307临时重定向
  • 40x:请求错误
    1. 400错误
    2. 401未授权
    3. 403禁止
    4. 404未找到
    5. 405方法禁用
    6. 406不接受
    7. 407需代理授权
    8. 408请求超时
    9. 409冲突
    10. 410已删除
    11. 411需要有效长度
    12. 412未满足条件
    13. 413请求实体过大
    14. 414请求URL过长
    15. 415不支持的媒体类型
    16. 416请求范围不合要求
    17. 417未满足期望值
    18. 428请求先决条件
    19. 429太多请求
    20. 431请求字段太大
  • 50x:服务器错误
    1. 500服务器内部错误
    2. 501尚未实施
    3. 502错误网管
    4. 503服务不可用
    5. 504网关超时
    6. 505HTTP版本不受支持
    7. 511要求网络认证

  • GET,POST,PUT,DELETE,PATCH
