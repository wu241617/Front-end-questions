常见的HTTP请求头

  • Accept：浏览器接受的格式
  • Accept-Encoding：浏览器接受的编码方式
  • Accept-Language：浏览器接受的语言，用于服务器判断多语言
  • Cache-Control：控制缓存的时效性
  • Connection：连接方式，keep-alive且服务端支持，则会复用TCP连接
  • Host：HTTP访问使用的域名
  • if-Modeified-Since：上次访问时的更改时间，如果服务端认为此时间后没有更新，则会给出304响应
  • if-None-Match：上次访问使用的E-Tag，通常是页面的信息摘要，比更改时间更准确一点
  • User-Agent：客户端标识，多数浏览器这个字段很复杂，包括操作系统，浏览器内核，版本号，HTML排版引擎等
  • Cookie：客户端存储的cookie字符串。
