CSS预处理器

  • 为CSS增加编程特性的拓展语言，可以使用变量，简单逻辑判断，函数等基本编程技巧。
  • CSS预处理器编译输出还是标准的CSS样式。
  • Less，Sass 都是动态的样式语言，是CSS预处理器。
  • Less变量符号@，Sass变量符号$。

CSS预处理器解决问题

  • CSS语法不够强大，因为无法嵌套导致有很多重复的选择器
  • 没有变量和合理的复用机制，导致逻辑上相关的属性值只能以字面量的形式重复输出，难以维护。

常用规则

  • 变量
  • 嵌套语法
  • 混入
  • @import
  • 运算
  • 函数
  • 继承

CSS预处理器带来的好处

  • CSS代码更加整洁，更易维护，代码量更少。
  • 修改更快，基础颜色使用变量。
  • 常用代码使用代码块，节省大量代码。
  • CSS嵌套减少了大量的重复选择器，避免一些低级错误。
  • 变量，混入大大提高了样式的复用性。
  • 更多的函数方法，让开发者能够有能力生成更加复杂的CSS样式。
