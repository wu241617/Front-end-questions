CSS伪类

  • 其核心就是用来选择DOM树之外的信息，不能被普通选择器选择的文档之外的元素，用来添加一些选择器的特殊效果。
  • :hover,:active,:visited,:link,:first0child,:focus,:lang等
  • 由于状态的变化是非静态的，所以元素达到一个特定状态时，它可能得到一个伪类的样式；当状态改变，它又会失去这个样式。

CSS伪元素

  • DOM树没有定义的虚拟元素
  • 核心就是需要创建通常不存在于文档中的元素。::after,::before

伪类和伪元素的区别

  • 表示方式不同：CSS3中，伪类:xxx,伪元素::xxx表示。
  • 定义不同：伪类：可以添加类来达到效果。伪元素：需要通过添加元素才能达到效果。
  • 相同之处：伪类和伪元素都不出现在源文件和DOM树中。
